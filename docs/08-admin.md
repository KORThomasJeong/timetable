# 08. 관리자 페이지

## 접속 방법

브라우저에서 `http://서버주소:3000/admin` 으로 접속합니다.

`ADMIN_PASSWORD` 환경변수가 설정된 경우 아이디/비밀번호를 묻습니다:
- **아이디**: `admin`
- **비밀번호**: `.env`의 `ADMIN_PASSWORD` 값

---

## 기능

### 1. 현재 설정 확인

페이지 상단에 현재 적용 중인 학교, 학년/반, 학교 코드를 표시합니다.

---

### 2. 학교 검색 및 선택

```
학교명 입력 → [검색] 버튼 → 결과 목록 → 학교 선택
```

1. 학교명을 입력합니다 (예: `홍천중`, `경기중`, `서울중`)
2. **검색** 버튼을 클릭합니다
3. 결과 목록에서 원하는 학교를 클릭해 선택합니다
   - 같은 이름의 학교가 여러 지역에 있을 수 있으므로 지역명을 확인하세요

> 💡 학교를 선택하면 내부 학교 코드(`school_code`)가 저장됩니다.
> 이후 시간표 수집 시 검색 과정을 건너뛰고 바로 해당 학교 데이터를 가져옵니다.

---

### 3. 학년/반 설정

- **학년**: 1 ~ 6 (중학교는 1~3)
- **반**: 1 ~ 20

---

### 4. 설정 저장

**[설정 저장]** 버튼을 누르면 학교 코드, 이름, 학년, 반이 DB에 저장됩니다.
다음 자동 업데이트(오후 7시)부터 새 설정이 적용됩니다.

---

### 5. 시간표 즉시 새로고침

**[시간표 새로고침]** 버튼을 누르면 저장된 설정으로 즉시 시간표를 수집합니다.
설정 변경 후 바로 확인하고 싶을 때 사용합니다.

> ⚠️ 수집에는 수 초~수십 초가 걸릴 수 있습니다. 버튼이 비활성화된 동안 기다려주세요.

---

## 인증 설정

### 비밀번호 설정하기

`.env` 파일에 추가:

```env
ADMIN_PASSWORD=원하는비밀번호
```

그리고 컨테이너 재생성:

```bash
docker compose up -d
```

### 비밀번호 없이 사용하기

`ADMIN_PASSWORD`를 설정하지 않으면 누구나 관리자 페이지에 접근할 수 있습니다.
**로컬 개발 환경에서만** 사용하고, 외부에 공개된 서버에서는 반드시 비밀번호를 설정하세요.

---

## API 직접 사용하기 (curl)

```bash
# 현재 설정 조회
curl -u admin:비밀번호 http://localhost:3000/admin/api/settings

# 학교 검색
curl -u admin:비밀번호 "http://localhost:3000/admin/api/search?q=홍천중"

# 설정 변경
curl -u admin:비밀번호 -X POST http://localhost:3000/admin/api/settings \
  -H "Content-Type: application/json" \
  -d '{"school_code":"20634","school_name":"홍천중학교","grade":"1","class_num":"3"}'

# 즉시 새로고침
curl -u admin:비밀번호 -X POST http://localhost:3000/admin/api/refresh
```
